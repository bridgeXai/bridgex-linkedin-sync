# BridgeX LinkedIn Sync · 快速参考卡

> 打印或钉在桌面旁，日常操作时对照使用。  
> 完整安装步骤见 **[安装指南.md](./安装指南.md)**

---

## 安装（一次性）

1. GitHub 下载 zip 或 `git clone` → 解压到固定目录
2. 向管理员索取 **`lib\secrets.js`** 放进 `lib\` 目录
3. `chrome://extensions/` → 开发者模式 → 加载已解压 → 选含 `manifest.json` 的文件夹
4. 选项页填：**操作者**（你的姓名）+ **LinkedIn 账户**（如 BRGX001）

---

## 每次操作前检查

- [ ] 当前 LinkedIn 账户与选项页 **LinkedIn 账户** 一致
- [ ] 在目标人 **个人主页**（不是搜索页 / 消息页）
- [ ] 已在 LinkedIn **手动**完成发连接 / 发消息

---

## 发连接

```
主页 → 📸 履历整页截屏（可选）→ 核对字段 → 已发申请
```

## 发私信（要 Pain/Offer 必须走这条）

```
主页 → 📸 履历整页截屏（可多次，内容会合并）
     → 或：页面划选 → 右键「同步此信息作为履历备注」
     → 粘贴消息全文到「发出的消息」
     → ✨ AI 分类
     → 已发消息
```

> **已发消息** 时消息框不能为空，否则不会同步。

## 连接被接受

```
同一人主页 → 连接已接受
```

---

## 字段从哪来？

| 想填的列 | 你要做什么 |
|----------|------------|
| career_background / edu_background | 📸 **履历整页截屏**；或划选/粘贴文本（自动 AI 解析） |
| message_sent | 粘贴**发出的消息** |
| pain_angle / offer_angle | 粘贴消息 → **✨ AI 分类** |
| operator_name | 选项页「操作者」 |
| account_id | 选项页「LinkedIn 账户」 |
| target_contact_email / target_contact_phone | 手动打开「联系方式」弹窗 → **📇 截屏识别** |

---

## 五个按钮

| 按钮 | 什么时候点 |
|------|------------|
| 已发申请 | 手动发完连接申请 |
| 已发消息 | 手动发完私信（须先粘贴消息） |
| 已回复 | 对方回复了你 |
| 连接已接受 | 对方通过了连接 |
| 其他 | 其它触达，记为 Sent |

---

## 出问题先看

1. Popup 底部 → **查看解析/同步记录**
2. 403 错误 → 联系管理员开飞书文档权限
3. Pain/Offer 空 → 是否粘贴了消息并点了 AI 分类？
4. 更新代码后没变化 → `chrome://extensions/` → **重新加载**
